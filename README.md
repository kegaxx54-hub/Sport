# Suivi Supplements & Sport

Petit site statique pour suivre:
- les prises de supplements chaque jour;
- les seances de sport;
- un debrief automatique en fin de mois.

## Utilisation en local

1. Ouvre `index.html` dans ton navigateur.
2. Selectionne ton mois.
3. Coche les cases au fur et a mesure.
4. Ajoute tes seances de sport.

Les donnees sont sauvegardees automatiquement dans le navigateur (localStorage).

## Mise en ligne avec GitHub Pages

1. Cree un nouveau repository GitHub (ex: `suivi-supplements-sport`).
2. Envoie ces fichiers dans le repo.
3. Va dans `Settings` > `Pages`.
4. Source: `Deploy from a branch`.
5. Branche: `main` et dossier `/ (root)`.
6. Sauvegarde et attends 1-2 minutes.
7. Ton site sera disponible sur une URL du style:
   `https://<ton-username>.github.io/suivi-supplements-sport/`
